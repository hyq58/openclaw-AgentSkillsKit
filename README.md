# 🛠️ OpenClaw Agent Skills Kit / OpenClaw 技能开发指南与模板库

Welcome to the **OpenClaw Agent Skills Kit**. This repository provides the community with best practices, templates, and examples for building robust, safe, and effective skills for OpenClaw agents.

欢迎来到 **OpenClaw Agent Skills Kit**。本仓库为社区提供构建稳健、安全、高效的 OpenClaw Agent 技能的最佳实践、模板和示例。

## Why This Matters / 为什么这很重要
OpenClaw's true power lies in its extensible skill system (`SKILL.md`). However, giving an AI agent tools without strict boundaries can lead to unpredictable behavior. This kit shows you how to write skills that agents understand perfectly and execute safely.

OpenClaw 的真正威力在于其可扩展的技能系统 (`SKILL.md`)。然而，如果不加严格限制地给 AI 提供工具，可能会导致不可预测的行为。本指南旨在教你如何编写能让 Agent 完美理解并安全执行的技能。

- 📘 **[SKILL-WRITING-GUIDE.md](SKILL-WRITING-GUIDE.md)**: The definitive guide to writing prompts, constraints, and tool definitions for agents. / 编写 Prompt、约束和工具定义的终极指南。
- 🏗️ **[templates/](templates/)**: Blank, boilerplate structures for your new skills. / 供你编写新技能使用的空白模板结构。

- 🧪 **[tools/](tools/)**: Scripts to lint and verify your `SKILL.md` files. / 用于静态检查和验证 `SKILL.md` 文件的脚本。
