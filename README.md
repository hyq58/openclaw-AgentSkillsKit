# 🛠️ OpenClaw Agent Skills Kit

Welcome to the **OpenClaw Agent Skills Kit**. This repository provides the community with best practices, templates, and examples for building robust, safe, and effective skills for OpenClaw agents.

## Why This Matters
OpenClaw's true power lies in its extensible skill system (`SKILL.md`). However, giving an AI agent tools without strict boundaries can lead to unpredictable behavior. This kit shows you how to write skills that agents understand perfectly and execute safely.

## Contents
- 📘 **[SKILL-WRITING-GUIDE.md](SKILL-WRITING-GUIDE.md)**: The definitive guide to writing prompts, constraints, and tool definitions for agents.
- 🏗️ **[templates/](templates/)**: Blank, boilerplate structures for your new skills.
- 💡 **[examples/](examples/)**: Production-tested examples (e.g., System Logger).
- 🧪 **[tools/](tools/)**: Scripts to lint and verify your `SKILL.md` files.
